# DBIT Alumni Website

Static multi-page alumni website for Don Bosco Institute of Technology, Bengaluru.

## Run locally

Open `index.html` directly in a browser, or serve the folder:

```sh
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Stack

- HTML5
- Tailwind CSS CDN
- Vanilla JavaScript
- Local scraped alumni image assets with source manifest
